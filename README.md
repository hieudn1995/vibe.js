# discord-music-bot [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]
> A simple music bot for Discord

## Installation

Clone this repository, and run:
```sh
$ npm install
```

## Usage

```sh
$ npm run bot
```
